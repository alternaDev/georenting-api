Auth

The GeoRenting API uses JSON Web Tokens for authentication. You will get one through the Sign In API Call. You need a Google Account Token to login. Authenticated API calls require the token you get after login to be sent in a Authorization header.

GCM

The API will leverage Google Butt Messaging to notify the user of specific events.

get /fences

Get Fences by user or around a center location. Returns more Information if the fence is owned by the current user.

  • Request
  • Response

Query Parameters

  • user: (string)

    Get all Fences owned by the specified User ID.

  • latitude: (number)

    Get Fences near the specified center location. Requires longitude and radius.

  • longitude: (number)

    Get Fences near the specified center location. Requires latitude and radius.

  • radius: (integer)

    Get Fences near the specified center location. Requires latitude and longitude.

  • excludeOwn: (boolean)

    Exclude GeoFences owned by the current user from the search. Requires Authorization, otherwise will be ignored.
